## Pontos não-óbvios pro próximo agente

1. **Não rastrear saldo de conta corrente nem criar tabela de dívidas separada** — registrado em memória e em BACKLOG. Se aparecer ideia "patrimônio líquido completo" ou similar, ver primeiro se cabe nas decisões já tomadas.
2. **HTTP cache `no-store`**: não remover — é mitigação ativa de bug Chrome com Apps Script Web App.
3. **PWA reload silencioso**: o `controllerchange` listener em main.tsx é interdependente com workbox `skipWaiting + clientsClaim`. Se mexer num, revisa o outro.
4. **Heurística de inferPessoaFromTitular**: match por `evellyn|eve|ivan|bam`. Se aparecer terceiro nome (caso raro), estender o regex ou criar coluna `aliases` na tabela `pessoas`.
5. **Samples de PDF**: `samples/` é gitignored — PDFs reais de fatura ficam só no machine do Ivan. Calibração de parser exige sample local.
6. **`useDialog` é o padrão canônico**: pra qualquer decisão tripartite (esta/futuras/cancelar), usar este hook. NÃO recriar o wrapper Promise-based em outras páginas.
